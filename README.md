## 📌 Sobre o projeto

Este repositório contém o código-fonte do meu portfólio pessoal, desenvolvido em **ReactJS**. O site apresenta minha trajetória profissional, experiências, habilidades e formas de contato.

> 💡 Este portfólio está disponível em **3 versões**, cada uma desenvolvida com uma tecnologia diferente:

| Versão       | Tecnologia              | URL                                                                |
| ------------ | ----------------------- | ------------------------------------------------------------------ |
| ✅ **Esta**  | ReactJS                 | [react.lucasoliveira04.com](https://react.lucasoliveira04.com)                 |
| 🔷 Angular   | Angular 19              | [angular.lucasoliveira04.com](https://angular.lucasoliveira04.com) |
| 🍃 Thymeleaf | Spring Boot + Thymeleaf | Em breve                                                           |

## ✨ Funcionalidades

- 🌍 **Internacionalização** — suporte a PT-BR, EN e ES via `i18next`
- 📱 **Responsivo** — adaptado para mobile, tablet e desktop
- ⚡ **Scroll suave** — navegação entre seções com offset do header fixo
- 📬 **Formulário de contato** — integração com serviço de e-mail
- 🔀 **Seletor de versão** — alterna entre as versões React, Angular e Thymeleaf

## 🚀 Como rodar localmente

```bash
# Clone o repositório
git clone https://github.com/lucasoliveira04/lucasoliveira04.com.git

# Entre na pasta
cd portfolio

# Instale as dependências
npm install

# Inicie o servidor de desenvolvimento
npm run dev
```

Acesse em `http://localhost:5173`
